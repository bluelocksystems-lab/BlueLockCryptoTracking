About

BlueLock Crypto Tracking
Version 1.4

Open-source cryptocurrency price and portfolio tracker. Built for privacy-conscious users who want a simple, transparent tool without accounts, logins, or third-party data sharing.

Supported Coins

⚠ Disclaimer

This application does not provide financial advice, custody services, wallet services, or transaction functionality.

  • Does NOT store private keys or seed phrases
  • Does NOT connect to any exchange accounts
  • Does NOT execute trades or send transactions
  • Does NOT hold any funds
  • Price data sourced from CoinGecko public API
  • All data stored locally on your device only

Tech Stack

Backend Python 3 + FastAPI
Frontend HTML + CSS + JavaScript
Database SQLite (local)
Prices CoinGecko Public API
Platform Windows 11
